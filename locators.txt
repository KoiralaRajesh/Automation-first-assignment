Go to the website: https://www.saucedemo.com/ and find the locators for the following elements:

 - Username input box
 - Password input box
 - Login button
 - Error message (when wrong login is attempted)


 1. Username Input Box
XPath: //input[@id='user-name']
CSS: #user-name

2. Password Input Box
XPath: //input[@id='password']
CSS: #password

3. Login Button
XPath: //input[@id='login-button']
CSS: #login-button

4. Error message (when wrong login is attempted)
XPath: //h3[@data-test='error']
CSS: h3[data-test='error']
